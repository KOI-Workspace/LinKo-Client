# 현수가 직접 작성한 기능명세서

## 자막에 표시되는 세가지 정보
1. 회색 하이라이트 박스: 이전에 대한 영상에서 북마크 집어넣은 경우
2. 보라색 박스 + 체크: 해당 영상의 플래시카드 레슨에서 북마크 집어넣은 경우
3. 보라색 글씨: 해당 영상의 플래시카드 레슨에서 북마크 안집어넣은 경우

## 단어 호버링 시 뜻
- 단어 위에 마우스를 호버링하면 뜻을 연결시켜주는 기능. 
- 사전 API 혹은 사전 데이터를 미리 저장해놨다가, 호버링하면 영어 뜻으로 표시해줘야 함. (어떻게, 어떤 사전 쓸지는 알아서 판단)

## 북마크
- 단어 호버링 시 북마크 할 수 있고, 문장도 옆에 있는 버튼 누르면 북마크 가능함. 
- 북마크 하면 북마크 페이지에서 확인 가능. 

## Cultural Notes
- 얘는 스크립트에서 속담, 한국어 슬랭같은 애들 뽑아서 만들어주면 됨. 
- 지난번에 너가 뽑아줬었던 문화 맥락 해설이 진짜 좋았음. 사실 그거랑 거의 동일해도 무방할 것 갗음.
